## ✅ Исправленные критические проблемы

### 1. ✅ Исправлена интеграция FormField в ProfileSelectionField

**Проблема:** `FormField` не обновлялся при изменении значения извне.

**Решение:**
- Преобразован `ProfileSelectionField` из `StatelessWidget` в `StatefulWidget`
- Добавлен `GlobalKey<FormFieldState<T>>` для управления состоянием
- Реализован `didUpdateWidget` для обновления значения `FormField` при изменении `value`

**Файл:** `profile_selection_field.dart`

---

### 2. ✅ Добавлена интеграция FormField в ProfileMultiSelectionField

**Проблема:** `ProfileMultiSelectionField` не был интегрирован с `Form`, валидация не работала.

**Решение:**
- Преобразован в `StatefulWidget`
- Добавлен `FormField<List<T>>` с валидацией
- Добавлен параметр `validator` для поддержки валидации
- Исправлена утечка памяти: добавлен `dispose()` для `TextEditingController` в диалоге
- Обновление значения через `field.didChange()` при изменении выбора

**Файл:** `profile_multi_selection_field.dart`

---

### 3. ✅ Исправлен парсинг роста

**Проблема:** Рост парсился как `double`, хотя должен быть целым числом (сантиметры).

**Решение:**
- Изменен парсинг с `double.tryParse()` на `int.tryParse()?.toDouble()`
- Сохранена совместимость с типом `double?` в `UserProfile`

**Файл:** `profile_edit_screen.dart:773`

**Было:**
```dart
height: _heightController.text.isEmpty
    ? null
    : double.tryParse(_heightController.text),
```

**Стало:**
```dart
height: _heightController.text.isEmpty
    ? null
    : int.tryParse(_heightController.text)?.toDouble(),
```

---

### 4. ✅ Реализована правильная прокрутка к ошибке

**Проблема:** Прокрутка всегда шла к началу (0), а не к первому полю с ошибкой.

**Решение:**
- Добавлены `GlobalKey` для каждой секции формы
- Реализована функция `_scrollToFirstError()` с использованием `Scrollable.ensureVisible()`
- Прокрутка происходит к первой секции, где могут быть ошибки
- Добавлена проверка `hasClients` перед использованием `ScrollController`

**Файл:** `profile_edit_screen.dart`

**Добавленные ключи:**
- `_personalInfoKey`
- `_physicalInfoKey`
- `_healthInfoKey`
- `_nutritionTargetsKey`
- `_goalsKey`
