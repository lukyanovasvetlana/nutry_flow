# Настройка Bottom Picker для выбора даты

**Дата:** 2025-01-27  
**Статус:** ⚠️ Требует установки пакета

---

## 📋 Шаги для установки

### 1. Установите пакет
Выполните в терминале:
```bash
flutter pub get
```

### 2. Раскомментируйте код
После успешной установки пакета:

1. **Раскомментируйте импорт** в `profile_edit_screen.dart`:
```dart
import 'package:bottom_picker/bottom_picker.dart';
```

2. **Раскомментируйте код Bottom Picker** в методе `_selectDateOfBirth`:
```dart
BottomPicker.date(
  title: 'Выберите дату рождения',
  // ... остальные параметры
).show(context);
```

3. **Закомментируйте или удалите** fallback на стандартный `showDatePicker`

## 📝 После установки

После выполнения `flutter pub get` и раскомментирования кода, Bottom Picker будет:
- Открываться снизу экрана
- Поддерживать формат dd/mm/yyyy
- Использовать тему приложения
- Иметь ограничения по датам (13-120 лет)
