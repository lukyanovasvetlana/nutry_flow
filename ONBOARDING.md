# 🚀 Подключение к Проекту NutryFlow

## Для новых разработчиков

### Шаг 1: Получите доступ к репозиторию

Обратитесь к Tech Lead или Scrum Master с вашим GitHub username для получения доступа.

### Шаг 2: Клонируйте репозиторий

```bash
git clone https://github.com/your-org/nutry_flow.git
cd nutry_flow
```

### Шаг 3: Запустите автоматическую настройку

```bash
chmod +x scripts/setup_developer_environment.sh
./scripts/setup_developer_environment.sh
```

### Шаг 4: Настройте переменные окружения

```bash
# Скопируйте шаблон
cp .env.example .env

# Отредактируйте .env и заполните реальные значения
# Получите их у Tech Lead или в документации проекта
```

### Шаг 5: Запустите приложение

```bash
flutter run
```
